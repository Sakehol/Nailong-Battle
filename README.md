## 🎮 游戏特色

- 🎤 **纯声控对战**：喊出短促的“哈！”来推动进度条，音量大、爆发力强就是力量！
- 🐲 **可爱奶龙主题**：内置两个超萌的奶龙角色（可自行替换 GIF）。
- ⏱️ **60 秒紧张倒计时**：超时自动按进度条位置判定胜负。
- 🤝 **再来一局友好协商**：双方同意后服务端统一重置，避免冲突。
- 💾 **战绩记录**：SQLite 自动存储对战历史，看看谁是真正的奶龙之王！
- 🖥️ **局域网联机**：一台电脑当主机，另一台直接加入，无需公网服务器。
- 🎨 **双缓冲绘制**：丝滑的拔河动画，零闪烁。

## 🚀 快速开始

### 主机（创建房间）

1. 双击运行 `VoiceBattle.exe`（即**奶龙大作战**）。
2. 登录或注册账号。
3. 在大厅点击 **🏠 创建房间（我是主机）**。
4. 界面会显示本机局域网 IP，**记下这个 IP**（例如 `192.168.1.100`）。
5. 将 IP 告诉另一位玩家，然后等待对方加入。

> 游戏会在第二位玩家加入时自动开始！

### 客机（加入房间）

1. 同样运行 `VoiceBattle.exe` 并登录。
2. 在大厅的 **服务器 IP** 输入框中填入主机提供的 IP 地址（如 `192.168.1.100`）。
3. 点击 **🔗 加入房间**。
4. 连接成功后自动跳转对战界面，准备嘶吼！

## 🔍 如何查找主机局域网 IP

1. 在主机电脑上按 `Win + R`，输入 `cmd` 回车。
2. 在黑色命令窗口中输入：

   ```cmd
   ipconfig
   ```

3. 找到当前使用的网络适配器（如“无线局域网适配器 Wi-Fi”），下面会显示 **IPv4 地址**。
4. 记下形如 `192.168.x.x` 的地址（注意：**不要使用 `127.0.0.1`**）。

> 💡 提示：如果主机有多张网卡（如虚拟机的虚拟网卡），请选择与你实际连接网络对应的那个 IPv4。

## 🔥 如何设置 Windows 防火墙允许游戏通信

主机必须放通 **TCP 端口 9527**，否则客机无法连接。

1. 按 `Win + R`，输入 `wf.msc` 回车，打开“高级安全 Windows Defender 防火墙”。
2. 左侧点击 **入站规则**，右侧点击 **新建规则…**。
3. 选择 **端口** → 下一步。
4. 选择 **TCP**，在“特定本地端口”输入 `9527` → 下一步。
5. 选择 **允许连接** → 下一步。
6. 勾选 **域**、**专用**、**公用** → 下一步。
7. 名称填 `奶龙大作战 TCP 9527`，完成。

> 测试时也可临时关闭防火墙，但测试完成后建议重新开启。

## ❓ 常见问题

**Q：客机输入 IP 后提示“连接失败”或长时间没反应？**  
A：  
- 确认两台电脑连在同一个路由器（Wi-Fi 名称一致）。  
- 检查主机防火墙是否放通了 9527 端口（或临时关闭防火墙）。  
- 确保主机已点击“创建房间”并保持在大厅界面。  
- 检查输入的 IP 是否为局域网地址（前三段与主机一致）。

**Q：喊“哈”没反应，进度条不动？**  
A：  
- 确认麦克风已接入且在系统中设为默认设备。  
- 观察对战界面右下角的**麦克风音量条**，如果没有跳动请检查麦克风。  
- 试着离近麦克风，用短促有力的“哈！”声，避免拖长音。  
- 如果环境嘈杂，可适当在代码中降低噪音阈值。

**Q：游戏结束后双方弹出了不同的结果？**  
A：  
- 新版已通过**服务端统一判胜负**修复，请确保使用最新版本。

**Q：怎么替换奶龙图片？**  
A：  
- 将你喜欢的 GIF 图片重命名为 `my_character.gif`（代表自己）和 `opponent_character.gif`（代表对手）。  
- 放入项目的 `Assets` 文件夹，并设置为**嵌入的资源**重新编译即可。  
- 不替换的话会显示默认的可爱圆形角色。

**Q：可以自己修改“哈”声的灵敏度吗？**  
A：  
- 可以。打开 `VoiceAnalyzer.cs`，调整 `HaThreshold`（默认0.12）和 `NoiseThreshold`（默认0.05）。  
- 数值越小越容易触发，但可能误触环境噪音。

## 🛠 技术栈

| 技术 | 说明 |
|------|------|
| **.NET 10** | Windows Forms 桌面框架 |
| **C# 13** | 主语言 |
| **NAudio 2.3.0** | 实时麦克风音频采集与分析 |
| **Entity Framework Core** + SQLite | 用户账号、对战记录存储 |
| **TCP 长连接 + 自定义协议** | 局域网对战网络通信 |
| **GDI+ 双缓冲绘制** | 流畅动画，无闪烁 |
| **ImageAnimator** | GIF 动画播放与重置 |

项目完全由 C# 实现，架构清晰，易于二次开发和客制化。
